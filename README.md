# Phaser game with planes and battles.
Demo on https://c301.github.io/planes-phaser-game/

How to run: 
-------------
- server index.html with http server
- open it in browser
- play

For example
`npm install -g http-server && http-server`, open address in browser

Controls
----------
- up arrow
- down arrow
- spacebar to fire
- mouse
